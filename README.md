   # Telegram-бот "Ингушский читатель" 📚
   
   Этот бот позволяет находить и скачивать книги ингушских авторов в формате PDF.
   
   ## Функционал:
   - Поиск книг по названию.
   - Поиск по ключевым словам.
   - Доступность версий на двух языках.
   
   ## Стек технологий:
   - Python 3
   - pyTelegramBotAPI
   - JSON (база данных)
